# N-Queens Problem Solver (Python)

This repository contains a Python implementation of the N-Queens Problem, a classic algorithmic challenge. It uses a backtracking algorithm to place N queens on an NÃ—N chessboard so that no two queens attack each other. The program displays each solution in a clear chessboard format and allows adjusting the board size.
